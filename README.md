# Cosmic Cruises
A website for future tourists to discover the solar system.

## Tech Stack

- Golang
- Go Fiber
- HMTX
- daisyUI
- TailwindCSS

## Instructions to Run
Clone Repo

In the project directory run:

`$ go run main.go`

Site will be up at localhost:3000
